OpenAI 縮減 Codex 模型上下文大小，從 372k 降至 272k

人工智慧研究機構 OpenAI 近期宣布，針對其 Codex 模型進行了關鍵調整，將模型的上下文（Context Size）處理能力，從原本的 372,000 個 token 縮減至 272,000 個 token。這項變動代表 Codex 模型在單次處理任務中，能納入分析或生成的程式碼及相關資訊量將會減少。此舉可能會影響仰賴 Codex 進行長篇程式碼生成或複雜任務的開發者，需要重新評估其應用策略，以適應模型新的處理限制。
